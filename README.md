# Frontend для сайта c портфолио Гордея

## Установка пакетов
```
npm i
```

### Запуск webpack сервера для разработки
```
npm run serve
```

### Сборка для продакшена
```
npm run build
```

### Запуск тестов (когда они будут :))
```
npm run test:unit
```

### Запуск линтера
```
npm run lint
```

### Запуск линтера с автофиксом
```
npm run lint:fix
```
